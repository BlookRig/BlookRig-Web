BlookRig

Take Blooket Raids to the Next Level.

What is BlookRig?

BlookRig is a program that actively searches for active Blooket game codes and automatically raids them with open-sourced cheating scripts. It's designed to emulate traditional cryptocurrency mining software in terms of visual and technical designs.

How does it work?

The program works by connecting to existing Blooket Nodes containing a list of BSIDs (Blooket Session ID) or connecting via your own BSID. Once connected to a valid BSID, BlookRig will access Blooket's Game ID API (without the BSID, Blooket will deny access to the API) and generate a series of 6-digit numbers, using the API to verify it's existence. Finally, the program will use open-sourced scripts from other developers to join a session and utilize cheating scripts to make it unplayable.

What's the benefit?

Our goal for BlookRig to optimize the user experience and utilization of exploitation on Blooket, introducing a new and automated way to disrupt Blooket games for people who wish to do so.
